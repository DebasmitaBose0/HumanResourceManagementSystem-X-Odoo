
# 👨‍💼 Human Resource Management System (HRMS)

A modern Human Resource Management System (HRMS) designed to simplify and automate day-to-day HR operations. The system enables organizations to efficiently manage employees, attendance, leave requests, payroll visibility, and administrative workflows through a secure role-based platform.

---

## 📌 Overview

The Human Resource Management System digitizes essential HR processes, providing separate dashboards for Employees and HR/Admin users. It improves operational efficiency, reduces manual work, and offers a centralized platform for employee management.

---

## ✨ Features

### 🔐 Authentication & Authorization
- Secure Sign Up and Sign In
- Email Verification
- Role-Based Access Control
- Password Security Validation

### 👤 Employee Management
- View Personal Profile
- Update Contact Information
- Upload Profile Picture
- View Job Details
- Access Salary Structure
- Document Management

### 📊 Dashboard

#### Employee Dashboard
- Personal Profile
- Attendance Overview
- Leave Requests
- Recent Activities
- Quick Navigation

#### Admin Dashboard
- Employee Management
- Attendance Monitoring
- Leave Approval
- Payroll Management
- Organization Overview

### 📅 Attendance Management
- Daily Attendance
- Weekly Attendance
- Check-In / Check-Out
- Attendance Status
  - Present
  - Absent
  - Half-Day
  - Leave

### 📝 Leave Management
- Apply for Leave
- Calendar-Based Date Selection
- Multiple Leave Types
  - Paid Leave
  - Sick Leave
  - Unpaid Leave
- Track Request Status
  - Pending
  - Approved
  - Rejected

### 💰 Payroll Management
- Employee Salary View (Read Only)
- Admin Salary Management
- Payroll Updates
- Salary Structure Management

## 👥 User Roles

### 👨‍💼 Admin / HR
- Manage Employees
- View All Attendance Records
- Approve or Reject Leave Requests
- Manage Payroll
- Update Employee Details
- Monitor HR Activities

### 👨‍💻 Employee
- View Personal Profile
- Update Limited Profile Information
- Mark Attendance
- Apply for Leave
- View Attendance History
- View Payroll Details
